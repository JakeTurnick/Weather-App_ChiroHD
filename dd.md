# Weather Forecast app

### 1.0 goals
- Frontend
    - searchbar for locations (name / zip)
    - sends to API and recieves forecast data
    - neatly displays forecast data
- Backend
    - API that takes requests, proxies to 3rd party API to get forecast
    - API Key is secured (env var)
- README
    - Neatly describe the project & technologies used

- Bonuses
    - Unit testing
    - User auth
    - Deployed backend with AWS Lambda + API Gateway
